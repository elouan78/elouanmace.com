Responsive Design

Je propose de rendre le contenu de votre site ou application web totalement dynamique en fonction de la taille de votre écran. ( Version Mobile, Tablette et Ordinateur )

SEO

L'optimisation du référencement d'un site internet ou d'une application web est l'une des choses les plus importantes. C'est pourquoi je travaille avec les bonnes pratiques du web pour permettre à mes clients de disposer du meilleur classement sur les moteurs de recherche. ( naturellement )

CMS

Je propose de créer votre site personnel comme professionnel sous plusieurs CMS ( Système de gestion de contenu) dont Wordpress et Shopyfy.

Maquettes

Avant le début de tout projet je prends en considération la totalité de vos besoins, je crée un cahier des charges et des maquettes graphique de votre futurs projets web.

Technologies

J'utilise au quoditien les technologies suiviante : HTML, CSS ( boostrap ), PHP ( symphony ), JavaScript ( jQuery, React ) ainsi que du SQL. Je m'adapte bien entendu à vos besoins.

Page d'administration

Si vous optez pour un site web ou une application web sans CMS ( Système de gestion de contenu) je vous propose pour autant un system de connexion administrateur qui vous permet de gérer votre projet web facilement sans savoir coder.
